Modules¶

The main HnTool program (core.py) runs a list of rules defined in __files__ and __services__.

  • __files__ :

    defines the rules which process simple files and configs.

  • __services__ :

    defines the rules which checks the security on services and daemons.

Once your module is finalized, remember to add it to the appropriate array (__files__ or __services__) defined in hntool/__init__.py
